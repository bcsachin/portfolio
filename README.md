# B C Sachin Portfolio

A resume-based single-page portfolio for B C Sachin, showcasing professional experience, technical skills, and selected engineering projects.

## Files
- `index.html` — custom portfolio content built from resume details
- `styles.css` — modern dark theme, glassmorphism, and motion styling
- `script.js` — scroll reveal and mobile menu interaction

## Usage
1. Open `index.html` in your browser.
2. Review experience, skills, and projects sections.
3. Update contact details or add additional project links if desired.

## Notes
- The portfolio highlights core strengths in React, DevOps, cloud platforms, and monitoring.
- It is designed to make your engineering background clear and compelling to recruiters, hiring managers, and technical leaders.
